# 航南生存模拟器：气味危机 v2

更新内容：

1. 每天第 3 回合固定进入宿舍，保证每天至少有一段宿舍场景。
2. 所有道具初始数量改为 1。
3. 增强负面状态影响，更容易死亡。
4. 新增非消耗道具恢复选项：离场透气、闭眼缓冲、自己洗澡。
5. 文案压缩，更适合快节奏点击。
6. 从“大场景”改为“具体人物画像”，每回合出现一个气味角色。

## 部署

将以下文件上传到 GitHub 仓库根目录：

- index.html
- style.css
- script.js
- README.md

然后在 Settings → Pages 中选择 main / root。


## v4 更新

- 每个场景只显示 4 个行动按钮。
- 不同场景显示不同按钮，例如：
  - 教室：忍了 / 开窗 / 委婉提醒 / 离场透气
  - 电梯：戴口罩 / 闭眼缓冲 / 直接开怼 / 薄荷糖
  - 宿舍：自己洗澡 / 洗衣液 / 宿舍公约 / 委婉提醒
  - 小组讨论：委婉提醒 / 直接开怼 / 戴口罩 / 离场透气
- 按钮布局改为 2×2，更清爽。


## v5 更新

- 每个新场景随机抽取 4 个可用行动按钮。
- 不再按场景固定显示四个选项。
- 每回合按钮会随机，但本回合内不会因状态刷新而反复跳动。
- 为了保证可玩性，随机池中会尽量保底：
  - 至少出现一个恢复/缓冲类选项；
  - 至少出现一个互动/处理类选项。
