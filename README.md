# 航南生存模拟器：四文件手机紧凑弹窗版

这是重新拆分后的四文件版本，适合 GitHub Pages 部署。

## 文件

- index.html
- style.css
- script.js
- README.md

## 本版功能

- 手机优先布局。
- 状态栏移到选项下面。
- 状态栏缩小并可折叠。
- 每回合只随机出现 4 个可用选项。
- 每回合结果单独弹窗显示，点击“继续”后进入下一回合。
- 每天第 3 回合固定宿舍。
- 所有道具初始数量都是 1。

## 上传方式

把这 4 个文件全部上传到 GitHub 仓库根目录并覆盖旧文件。

如果 GitHub Pages 没立刻更新：

1. 等 1–3 分钟；
2. 打开网页后按 Ctrl + F5；
3. 或者在网址后面加 `?v=mobile2`。


## easy1 平衡性更新

- 降低整体气味强度与场景难度。
- 降低被动掉血、掉空气、掉忍耐、掉理智的速度。
- 降低电梯和宿舍额外伤害。
- 降低随机事件的负面冲击。
- 减少“忍了”“开窗”“委婉提醒”“直接开怼”等行动的副作用。
- 每日结束后的恢复略增强。
