# 航南生存模拟器：气味危机 V6 Clean

这是从零重写的干净版本，不再沿用旧代码。

## 本版保证

- 每回合只显示 4 个选项。
- 4 个选项是随机抽取的，不按场景固定。
- 第 3 回合固定宿舍，因此每天至少一次宿舍。
- 所有道具初始数量都是 1。
- 文案更短。
- 每回合显示具体人物形象。

## 文件

- index.html
- style.css
- script.js
- README.md

## 上传 GitHub Pages

把这 4 个文件直接放到仓库根目录，不要套文件夹。
然后进入 Settings → Pages，选择 main / root。
